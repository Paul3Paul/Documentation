Introduction

HTML stands for HyperText Markup Language. It is used to design web pages using a markup language. HTML is a combination of Hypertext and Markup language. Hypertext defines the link between web pages. A markup language is used to define the text document within the tag which defines the structure of web pages. This language is used to annotate (make notes for the computer) text so that a machine can understand it and manipulate text accordingly. Most markup languages (e.g. HTML) are human-readable. The language uses tags to define what manipulation has to be done on the text.

HTML is a markup language used by the browser to manipulate text, images, and other content, in order to display it in the required format. HTML was created by Tim Berners-Lee in 1991. The first-ever version of HTML was HTML 1.0, but the first standard version was HTML 2.0, published in 1995

Structure of an HTML

The structure of an HTML document follows a specific format. Here's a basic outline of the structure of an HTML document:

Let's break down each section :

  1. !DOCTYPE html: This declaration defines the document type and version of HTML. In modern web development, !DOCTYPE html is used for HTML5.

  2. <html> : The root element of the HTML document. The lang attribute is used to specify the language of the document.
  3. <head>: This section contains meta-information about the HTML document, such as character set, viewport settings, and the document's title. It also includes links to external stylesheets and scripts.
  4. <meta charset="UTF-8">: Defines the character set for the document. UTF-8 is widely used and supports a broad range of characters.
  5. <meta name="viewport" content="width=device-width, initial-scale=1.0">: This meta tag is important for responsive web design. It ensures that the webpage adapts to the screen size of different devices.
  6. <title>: Sets the title of the HTML document, which appears in the browser tab or window.
  7. <body>: This is where the main content of the HTML document goes. It includes headings, paragraphs, images, links, forms, and other HTML elements that make up the visible part of the webpage.
  8. Additional Content: Within the '<head>' and <body> sections, you can include other HTML elements, such as stylesheets (<link>), scripts (<script>), and additional meta tags.
  • It's important to note that this is a basic structure, and real-world web pages often include additional elements and attributes for styling (using CSS) and interactivity (using JavaScript). CSS and JavaScript are usually linked or embedded in the HTML document to enhance the visual appearance and functionality of the webpage.

Tags

HTML tags are the building blocks of an HTML document. They define the structure and content of a web page by marking different elements. HTML tags are always enclosed in angle brackets (<>), and most come in pairsâ€”a start tag and an end tag. Here are some common HTML tags:

  • !DOCTYPE html>:< Declaration specifying the document type and version of HTML.
  • <html>: The root element of an HTML page.
  • <head>: Contains meta-information about the HTML document, such as title, character set, styles, and scripts.
  • <title>: Sets the title of the HTML document, which appears in the browser's title bar or tab.
  • <body>: Contains the main content of the HTML document, including text, images, links, and other elements.
  • Headings (<h1>, <h2>, <h3>, <h4>,<h5>, <h6>): Used to define headings of different levels.
  • Paragraph (<p>):Defines a paragraph of text.
  • Link :(<a>) Creates hyperlinks to other web pages or resources.
  • Image (<img>): Embeds images in the document.
  • List (<ul>, <ol>, <li>): Defines unordered or ordered lists.
  • Division (<div>): Groups content together and is often used for styling with CSS.
  • Span(<span>): Inline container used for styling specific portions of text.

There are many more tags available for various purposes. Additionally, attributes can be added to tags to provide extra information or control their behavior. Understanding HTML tags is fundamental for creating web pages and understanding web development.

Features

Features of HTML

  • It is easy to learn and easy to use.
  • It is platform-independent.
  • Images, videos, and audio can be added to a web page.
  • Hypertext can be added to the text.

Advantages

  • HTML is used to build websites.
  • It is supported by all browsers.
  • It can be integrated with other languages like CSS, JavaScript, etc.

Disadvantages

  • HTML can only create static web pages. For dynamic web pages, other languages have to be used.
  • A large amount of code has to be written to create a simple web page.
  • The security feature is not good.
